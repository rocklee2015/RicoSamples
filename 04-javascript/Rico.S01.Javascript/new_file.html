在运营工作中，不管是用户运营、内容运营，还是其他各种名头的运营人员，可能或多或少都会接触到做活动，而一个小小的简单的活动帖，其实写起来也有很多细节，很多容易忽略的地方需要去注意，去优化。而一个活动是否成功，也不仅仅是你的文案写的有多华丽，奖品有多丰厚，活动的帽子扣的有多大就能决定的。

    太多的运营人员做活动就仅仅止步于活动上线，而忽略活动中的数据观察，活动后的反馈总结。下面的内容，算是我每次做完一个小活动后，都会给自己记录的一些要考虑、要改进的地方，慢慢的就变成了这样。个人觉得这些内容，算是做活动必须要考虑的最基本的啦。它们很小、很碎、也很容易忘记~~是我工作的记录，也希望对你有帮助！
