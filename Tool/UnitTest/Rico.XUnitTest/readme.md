
##### XUnit
- Xunit其实是JUnit的衍生版
> xunit (原名为xunit.net)  主要
> xunit.runner.console  运行结果写在控制台
> xunit.runners
> xunit.runners.visualstudio  和VS测试管理器结合使用

##### 注意
> 1. 2.0版本以后不支持 控制台输出，而且多了两个依赖项
